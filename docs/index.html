Features

Real-time LiDAR

실시간 2D LiDAR 데이터를 웹 브라우저에서 시각화합니다. 성능 최적화로 부드럽게 동작합니다.

Joystick Control

nipplejs 기반 터치 조이스틱으로 모바일/태블릿에서 직관적인 로봇 제어가 가능합니다.

SLAM Mapping

Cartographer SLAM으로 실시간 지도 생성. 웹에서 바로 맵핑 진행 상황을 확인합니다.

Navigation2

맵에서 드래그로 목적지와 방향을 직관적으로 설정하여 자율 주행을 시작합니다.

Camera View

160도 어안카메라 영상을 실시간으로 스트리밍합니다. 풀스크린 지원.

Offline Ready

모든 JS 라이브러리를 로컬에 저장하여 인터넷 없이도 동작합니다.
